### 3. 管理画面ログイン (/admin/login)
- [ ] ログインページが表示される
- [ ] メールアドレス入力フィールドがある

- [ ] Email: `wrong@example.com`
- [ ] Password: `wrongpass`
- [ ] エラーメッセージが表示される

#### 有効な認証情報でのテスト
- [ ] Email: [.envファイルのNEXT_PUBLIC_ADMIN_EMAIL]

- [ ] ログインが成功する
- [ ] 管理画面にリダイレクトされる

### 4. 管理画面 (/admin)
- [ ] 未ログイン状態でアクセス → ログインページにリダイレクト
- [ ] ログイン後にアクセス → 管理画面が表示される
- [ ] メニュー管理インターフェースが表示される
- [ ] データが読み込まれている

#### メニュー管理
